Calculator Numerical
====================

三角函数使用弧度制单位。

disigned by Anne Wu 2018.12.13

---
# 功能说明
### *程序实现了基本的四则运算计算器功能：*
> 	支持运算符优先级，先乘除后加减；
> 	支持多重嵌套的括号；
> 	支持针对整数和浮点数的运算
> 	具备足够的健壮性，当用户输入非法的表达式时，能继续工作并给出提示；
> 	具有历史记录功能，能对之前的计算过程进行回滚。
> 	支持部分三角函数，部分其他运算符号，比如开根阶乘等
---
